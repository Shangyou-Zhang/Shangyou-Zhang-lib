
# 一、算术运算符:

算术运算符用于执行基本的数学运算，包括加法、减法、乘法、除法、取余、自增和自减。

|运算符|描述|示例|
|---|---|---|
|`+`|加法|`a + b`|
|`-`|减法|`a - b`|
|`*`|乘法|`a * b`|
|`/`|除法|`a / b`|
|`%`|取余|`a % b`|
|`++`|自增|`a++` 或 `++a`|
|`--`|自减|`a--` 或 `--a`|

**示例：**

```c
#include <stdio.h>

int main() {
    int a = 10, b = 5;
    int sum = a + b;
    int diff = a - b;
    int product = a * b;
    int quotient = a / b;
    int remainder = a % b;
    int increment = ++a; // 或 a++
    int decrement = --b; // 或 b--

    printf("加法：%d + %d = %d\n", a, b, sum);
    printf("减法：%d - %d = %d\n", a, b, diff);
    printf("乘法：%d * %d = %d\n", a, b, product);
    printf("除法：%d / %d = %d\n", a, b, quotient);
    printf("取余：%d %% %d = %d\n", a, b, remainder);
    printf("自增：%d\n", increment);
    printf("自减：%d\n", decrement);

    return 0;
}
```

**输出：**

```
加法：10 + 5 = 15
减法：10 - 5 = 5
乘法：10 * 5 = 50
除法：10 / 5 = 2
取余：10 % 5 = 0
自增：11
自减：4
```

**注意事项：**

- **除法运算：** 在整数除法中，结果会被截断为整数部分。
- **取余运算：** `a % b` 返回 `a` 除以 `b` 的余数。
- **自增和自减：** `++a` 或 `--a` 为前置自增/自减，`a++` 或 `a--` 为后置自增/自减。前置和后置的主要区别在于表达式的求值顺序。

# 二、关系运算符：

关系运算符用于比较两个值的大小或相等性，结果为布尔值（在 C 语言中，`0` 表示假，非零值表示真）。

## 关系运算符不可以连续判断 ~~3<a<10~~中间应该用逻辑运算符连接

|运算符|描述|示例|
|---|---|---|
|`==`|等于|`a == b`|
|`!=`|不等于|`a != b`|
|`>`|大于|`a > b`|
|`<`|小于|`a < b`|
|`>=`|大于等于|`a >= b`|
|`<=`|小于等于|`a <= b`|

**示例：**

```c
#include <stdio.h>

int main() {
    int a = 10, b = 5;

    if (a > b) {
        printf("%d 大于 %d\n", a, b);
    }

    if (a != b) {
        printf("%d 不等于 %d\n", a, b);
    }

    return 0;
}
```

**输出：**

```
10 大于 5
10 不等于 5
```

**注意事项：**

- **优先级：** 在表达式中，算术运算符的优先级高于关系运算符。
- **类型转换：** 当比较不同类型的值时，C 语言会进行隐式类型转换。
